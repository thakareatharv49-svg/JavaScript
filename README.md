# 🚀 DOM Day 10 - To-Do App

## 📖 Overview

This project is the final project of my 10-Day JavaScript DOM Learning Roadmap.

I built a fully functional To-Do App using Vanilla JavaScript without any libraries or frameworks.

---

## ✨ Features

- ➕ Add new tasks
- 📝 Display tasks dynamically
- ❌ Delete tasks by clicking on them
- 🧹 Automatically clear the input field
- 🚫 Prevent empty tasks from being added

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation

---

## 📚 Concepts Practiced

- DOM Selection (`querySelector`)
- Event Listeners
- Input Handling (`.value`)
- `createElement()`
- `appendChild()`
- `remove()`
- `innerText`
- Dynamic DOM Manipulation
- Form Validation
- Functions

## 💻 JavaScript Flow

```
User enters a task
        │
        ▼
Click "Add Task"
        │
        ▼
Read input value
        │
        ▼
Create <li> element
        │
        ▼
Set task text
        │
        ▼
Attach delete event
        │
        ▼
Append task to list
        │
        ▼
Clear input field
```
